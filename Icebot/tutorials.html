Icebot Documentation

Bot to combine Dyno & Mee6

Description

Icebot is a Discord bot created by Charley/Syntaxis @ SyntaxisCS and is meant to serve as a multipurpose bot taking inspiration from commands found in Dyno. Icebot also comes packaged with a leveling system that is meant to be a more advanced leveling algorithm than Mee6.

Installation

Icebot can be added to your server by clicking the invite button on the home page of icebot.syntaxiscs.com

Commands

Info

All commands under this category relate to retrieving information regarding Icebot

*help

Help allows users to look at the list of commands offered by Icebot in nicely serparated menus

*botinfo OR *info

BotInfo/Info is one of many commands in Icebot that has an alias, meaning that users can type either command and it will return the same thing. Info returns an embed with bot info like it's version, creator, server hosting, uptime, etc

*ping

Pong! Ping is a very simple command to allow users to see bot status or latency

*uptime

Uptime is a command that shows the uptime in the bot, this information can also be seen in the info command but has it's own seperate command for ease of use

*prefix

A prefix command is a must have of any bot that allows server owners to change its prefix. Prefix is a command that can either be called with the default prefix of * or with whatever the owner has chosen. This allows a user to see what the prefix is in any server if the owner doesn't have it clearly shown

Fun

All commands under this category relate to users curing their boredom and having fun with their friends

*coinflip

Coinflip is a command to be used when you really can't decide on anything. Very simple, Heads or Tails!

*rockpaperscissors OR *rps

Rockpaperscissors or shortened to rps is the age-old game in Discord form. At launch you can only play against the computer but plans to play against another user is planned after release

*bored

Bored is the first command that is used to help cure your boredom. It uses an api to give you a random task to do or game to play with either yourself or up to 8 people. Try it out!

*covid19 OR *covid19 country (query) OR *covid19 state (query)>

Covid19 is a command that every wished wasn't a thing but because of the current global pandemic I have added this command to Icebot for the foreseeable future. It can be used to see global, country, or US state covid19 stats

*welcome

The welcome command is a command that is connected to our welcoming system. Welcome allows any user to see what their welcome card looks like without having to leave and rejoin the server. That would get annoying, right?

*profile

Profile is a command that is connected to the leveling system. This allows a user to see what level they are as well as how far they are through the current level and how far they have until the next.

*leaderboard

Leaderboard allows users to see the top 10 users sorted by xp in the server

Moderation

The following commands are only able to be called by moderators or people with advanced permissions in servers

*kick (@mention)

Kicks the specified user. Must have the KICK MEMBERS permission

*ban (@mention)

Bans the specified user. Must have the BAN MEMBERS permission

*purge (#)

Allows users with the MANAGE MESSAGES permission to delete up to 100 messages at a time that were sent in the last 2 weeks. Discord API limitations unfortunatly :/

*setprefix (prefix)

Changes the prefix that Icebot will respond to in the server. This command can only be called by admins

*setwelcome (channel-name)

Changes the channel where Icebot will send the welcome cards for every new user. This must also be called by an admin

*welcometoggle

Changes whether or not Icebot will welcome new members. Please insure you use the above command to make sure this works correctly. This command can only be called by admins

*setleveling

Turns on or off the leveling system. This command can only be called by admins
