Icebot Documentation

Bot to combine Dyno & Mee6

Description

Icebot is a Discord bot created by Charley/Syntaxis @ SyntaxisCS and is meant to serve as a multipurpose bot taking inspiration from commands found in Dyno. Icebot also comes packaged with a leveling system that is meant to be a more advanced leveling algorithm than Mee6.

Commands

Info

All commands under this category relate to retrieving information regarding Icebot

*help

Help allows users to look at the list of commands offered by Icebot in nicely serparated menus

*botinfo OR *info

BotInfo/Info is one of many commands in Icebot that has an alias, meaning that users can type either command and it will return the same thing. Info returns an embed with bot info like it's version, creator, server hosting, uptime, etc

*ping

Pong! Ping is a very simple command to allow users to see bot status or latency

*uptime

Uptime is a command that shows the uptime in the bot, this information can also be seen in the info command but has it's own seperate command for ease of use

*prefix

A prefix command is a must have of any bot that allows server owners to change its prefix. Prefix is a command that can either be called with the default prefix of * or with whatever the owner has chosen. This allows a user to see what the prefix is in any server if the owner doesn't have it clearly shown
